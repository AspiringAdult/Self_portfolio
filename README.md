## 🧠 Concept

Instead of listing projects statically, this portfolio transforms them into **“mission files” inside immersive environments**.

Each section is a **world**:

- 🦇 **Shadow Intelligence** → AI systems & RAG pipelines  
- 🕸️ **Web of Systems** → Algorithmic trading & data flows  
- 🔐 **Cyber Defense Zone** → VPN architecture & security  
- 📊 **Data Lab** → Visualization & analytics  
- 🧠 **Core Identity Chamber** → Personal system overview  

The goal:  
> Let users *experience how systems think*, not just read about them.
